This is a beginner-friendly Java + Hibernate mini-project that combines a simple 
Tic-Tac-Toe (XO) game with database storage. Players play the XO game in the console,
and the winner details are stored in a MySQL database using Hibernate ORM.


## Features

### Game Features
- Two-player Tic-Tac-Toe (Player X & Player O)
- Console-based interface
- Automatic winner detection
- Supports Draw results

### Hibernate + Database Features
- Stores each game's result:
  - Player X name
  - Player O name
  - Winner (X, O, or Draw)
  - Date & Time of game
- View full match history from database
- Uses Hibernate ORM for easy Java-to-database mapping

## Technologies Used
- Java
- Hibernate ORM
- MySQL
- JPA Annotations
- Console-based UI
